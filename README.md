# JavaScript Langua Basics

# Variables
In strict mode, variable configuration comprises of 3 phases

- Declaration
- Assignment
- Initialization
  
Declaration is defining scope and name for varaible
- var userName;
  
Assignment is rendering a value into variable after declaring
  - userName = 'arjun';

Initialization is rendering a value into variable while declaring
- var userName = 'arjun';
---

# Var

Var is function scope Variable.

We can declare in any block of function & access it from any other block function.

---

var allows shadowing.

Shadowing is process of redeclaring same name identifier within the scope.

---

var also allows hoisting.

It is a compiling technique, where compiler can find the declaration of variable before using it.

---

# let

let defines block scope variable.

It is accessible only in block, where it is declared/assign/initialize or to its inner block.

It won't allow hosting.

It won't allow shadowing.

---

# const

const is also used to define block scope variable.

It will allow only Initalization.

It neither allow declaration nor assignment. 

More restricted than var & let.
